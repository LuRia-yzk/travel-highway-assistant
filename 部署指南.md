## 🚀 快速开始

### 前置要求

- Node.js >= 14.0.0
- npm >= 6.0.0
- 微信开发者工具
- 高德地图 API Key

### 1. 本地开发（5 分钟）

```bash
# 克隆项目
cd travel-highway-assistant

# 安装后端依赖
cd backend
npm install

# 启动后端服务
node server.js

# 打开微信开发者工具
# 导入 frontend 目录
# 编译运行
```

### 2. 配置高德 API

编辑 `backend/config/index.js`:
```javascript
amap: {
  key: '你的 Web 服务 Key',
  webKey: '你的 Web 端 Key',
  securityCode: '你的安全密钥'
}
```

### 3. 测试

访问：http://localhost:3000/api/highway/all

## 🌐 生产环境部署

### 方案 A: 腾讯云 CloudBase（推荐）

**优势**:
- 免运维，自动扩缩容
- 与微信小程序深度集成
- 免费额度：500GB 存储，50 万次读/月
- 一键部署

**步骤**:
```bash
# 1. 安装 CLI
npm install -g @cloudbase/cli

# 2. 登录
tcb login

# 3. 创建环境
tcb env create -n highway-assistant

# 4. 部署云函数
tcb fn deploy getHighwayNodes --filePath backend/

# 5. 查看部署
tcb fn list
```

### 方案 B: 云服务器（ECS/CVM）

**服务器配置**:
- CPU: 2 核
- 内存：4GB
- 带宽：3Mbps
- 系统：Ubuntu 20.04 LTS

**部署步骤**:

```bash
# 1. 连接服务器
ssh root@your-server-ip

# 2. 安装 Node.js
curl -sL https://deb.nodesource.com/setup_18.x | bash -
apt-get install -y nodejs npm

# 3. 上传代码
scp -r backend/* root@your-server-ip:/var/www/highway-assistant

# 4. 安装依赖
cd /var/www/highway-assistant
npm install --production

# 5. 配置环境变量
cp .env.example .env
nano .env  # 编辑配置

# 6. 使用 PM2 管理进程
npm install -g pm2
pm2 start ecosystem.config.js
pm2 save
pm2 startup

# 7. 验证服务
curl http://localhost:3000/api/highway/all
```

### 方案 C: 一键部署脚本

```bash
# 下载脚本
wget https://raw.githubusercontent.com/your-repo/deploy.sh

# 赋予执行权限
chmod +x deploy.sh

# 运行脚本
sudo ./deploy.sh
```

## 🔒 HTTPS 配置

### 使用 Let's Encrypt 免费证书

```bash
# 安装 Certbot
apt-get install certbot python3-certbot-nginx

# 获取证书
certbot --nginx -d your-domain.com

# 自动续期
certbot renew --dry-run
```

### Nginx 配置

使用提供的 `deploy/nginx.conf` 模板:
```bash
cp deploy/nginx.conf /etc/nginx/sites-available/highway-assistant
ln -sf /etc/nginx/sites-available/highway-assistant /etc/nginx/sites-enabled/
nginx -t
systemctl restart nginx
```

### 微信小程序域名配置

1. 登录微信公众平台
2. 开发 → 开发管理 → 开发设置
3. 服务器域名 → request 合法域名
4. 添加：`https://your-domain.com`

## 🔧 常见问题

### 1. 后端服务无法启动

**检查端口占用**:
```bash
netstat -tunlp | grep 3000
kill -9 <PID>
```

**查看日志**:
```bash
pm2 logs highway-assistant
```

### 2. 小程序请求超时

**检查域名配置**:
- 确保使用 HTTPS
- 域名已备案
- 已在微信公众平台配置

**检查防火墙**:
```bash
ufw status
ufw allow 443/tcp
```

### 3. 位置获取失败

**前端配置**:
- 检查 `app.json` 中的 `requiredPrivateInfos`
- 模拟器需配置位置
- 真机需授权位置权限

**后端配置**:
- 检查高德 API Key 是否有效
- 检查安全密钥配置

### 4. PM2 服务开机自启

```bash
# 生成启动脚本
pm2 startup

# 保存当前进程
pm2 save

# 验证
pm2 list
```

### 5. 数据库文件路径

确保数据文件路径正确:
```javascript
// backend/config/index.js
database: {
  roadNodes: './database/roadNodes.json',
  pois: './database/pois.json'
}
```
