## 五、API 接口速查

### 高速公路相关

| 接口 | 方法 | 说明 | 示例 |
|------|------|------|------|
| `/api/highway/all` | GET | 获取所有路口 | `curl localhost:3000/api/highway/all` |
| `/api/highway/recommend` | GET | 推荐前方路口 | `?lat=39.9042&lng=116.4074` |
| `/api/highway/:id` | GET | 获取路口详情 | `/api/highway/1` |
| `/api/highway/nearby` | GET | 搜索附近路口 | `?lat=39.9&lng=116.4&radius=50000` |
| `/api/highway/:id/weather` | GET | 获取路口天气 | `/api/highway/1/weather` |

### POI 相关

| 接口 | 方法 | 说明 | 示例 |
|------|------|------|------|
| `/api/poi/exit/:exitId` | GET | 获取路口所有 POI | `/api/poi/exit/1` |
| `/api/poi/exit/:exitId/type/:type` | GET | 按类型获取 POI | `/api/poi/exit/1/type/hotel` |
| `/api/poi/nearby` | GET | 搜索附近 POI | `?lat=39.9&lng=116.4&type=hotel` |
| `/api/poi/:poiId` | GET | 获取 POI 详情 | `/api/poi/1` |

## 七、常见问题 FAQ

### Q1: 后端启动失败怎么办？

**A:** 检查端口是否被占用
```bash
# Windows
netstat -ano | findstr :3000
taskkill /PID <进程 ID> /F

# 然后重新启动
npm start
```

### Q2: 小程序无法连接后端？

**A:** 
1. 确保后端已启动（访问 http://localhost:3000/health）
2. 检查 `app.js` 中的 `baseUrl` 配置
3. 微信开发者工具 → 详情 → 本地设置 → 勾选"不校验合法域名"

### Q3: 位置获取失败？

**A:**
1. 小程序右上角菜单 → 设置 → 允许位置权限
2. 真机需要开启 GPS
3. 模拟器设置中启用位置模拟

### Q4: 高德 API 调用失败？

**A:**
1. 检查 `backend/config/index.js` 中的 Key 配置
2. 登录高德开放平台查看配额
3. 检查网络连通性

### Q5: 如何修改数据？

**A:**
- 路口数据：编辑 `database/roadNodes.json`
- POI 数据：编辑 `database/pois.json`
- 重启服务器使更改生效
