## 九、问题排查

### 常见问题及解决方案

#### 1. 后端启动失败
**症状：** 端口被占用  
**解决：**
```bash
# Windows
netstat -ano | findstr :3000
taskkill /PID <进程 ID> /F
```

#### 2. API 返回 404
**症状：** 路由不存在  
**解决：** 检查路由前缀 `/api/highway` 或 `/api/poi`

#### 3. 小程序无法连接
**症状：** request fail  
**解决：** 
- 检查后端是否启动
- 检查 baseUrl 配置
- 微信开发者工具中开启"不校验合法域名"

#### 4. 位置获取失败
**症状：** 权限被拒绝  
**解决：** 
- 小程序设置中允许位置权限
- 真机需要 GPS 开启
